FAQ

Straight answers based on the current build.

Is ZenShift available on iOS?

iOS billing is not implemented yet. Android is live on Google Play. You can request iOS updates via the contact CTA.

Can I generate custom sessions today?

Yes. The generation pipeline is live (script â†’ TTS â†’ music mix). Quota checks exist but are currently bypassed in code, so limits are not enforced.

What powers the AI coach?

Amazon Bedrock Nova Micro via LangChain and a conversation state graph. Safety chains and moderation run before and during responses.

How is my data stored?

Auth uses Cognito. Conversation state, history, and favorites sync through AppSync + DynamoDB with offline queueing. Audio is stored in S3 behind signed URLs.

Do you have analytics?

Client-side events are tracked locally; there is no server-side attribution or funnel analytics beyond CloudWatch metrics for backend jobs.
