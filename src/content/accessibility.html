One Ping Fan Venture LLC is committed to ensuring that the ZenShift Coach website and mobile application are accessible to all users, including those with disabilities. We strive to comply with the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standards.

Our Commitment

We believe that everyone should have equal access to our AI-powered meditation coaching platform. This commitment includes:

  • Providing clear, consistent navigation throughout our website and app
  • Ensuring sufficient color contrast for all text and interface elements
  • Supporting keyboard navigation for all interactive elements
  • Including meaningful alternative text for images and graphics
  • Designing for screen reader compatibility
  • Supporting browser zoom up to 200% without horizontal scrolling

Accessibility Features

Website

  • Keyboard Navigation: All interactive elements can be accessed using only a keyboard
  • Skip Links: Skip to main content links for screen reader users
  • Focus Indicators: Clear visual focus indicators for keyboard navigation
  • Color Contrast: All text meets WCAG 2.1 AA contrast requirements (4.5:1 minimum)
  • Responsive Design: Works across all device sizes and orientations
  • Semantic HTML: Proper heading structure and landmark roles

Mobile App

  • Screen Reader Support: Full compatibility with VoiceOver (iOS) and TalkBack (Android)
  • Dynamic Type: Support for system font size preferences up to 200%
  • Reduced Motion: Respects user preferences for reduced motion
  • Voice Control: Compatible with voice navigation features
  • High Contrast: Supports high contrast display modes

Ongoing Efforts

We continuously work to improve accessibility through:

  • Regular accessibility audits and testing
  • User testing with assistive technology users
  • Employee training on accessibility best practices
  • Incorporating accessibility considerations into our design and development process

Third-Party Content

While we strive to ensure accessibility across our platform, some third-party content (such as meditation audio from external providers) may not meet the same accessibility standards. We work with our content partners to improve accessibility whenever possible.

Feedback and Support

We welcome feedback about the accessibility of ZenShift Coach. If you encounter any accessibility barriers or have suggestions for improvement, please contact us:

  • Email: zenshift-support@onepingfanventure.com
  • Subject Line: "Accessibility Feedback"

We aim to respond to accessibility feedback within 2 business days and will work with you to address any issues.

Technical Specifications

ZenShift Coach relies on the following technologies to work with assistive technologies:

  • HTML5
  • CSS3
  • JavaScript (ES6+)
  • ARIA (Accessible Rich Internet Applications) attributes

Compliance Status

This website aims to conform to WCAG 2.1 Level AA. We recognize that accessibility is an ongoing effort and continue to identify and address areas for improvement.
